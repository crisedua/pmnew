# Product Overview

**Portal de Proyectos** is a comprehensive project management platform with a modern interface, Kanban-style task tracking, document management, and team collaboration features.

## Core Features

- **Project Management**: Visual project overview with progress tracking and statistics
- **Task Management**: Kanban board and list views for task organization
- **Document Management**: File storage and organization system
- **Team Collaboration**: Team member management and task assignment
- **Real-time Sync**: Supabase integration for live data updates
- **Responsive Design**: Adaptive interface for all devices

## User Experience

- **Dual Theme**: Light theme for landing page, dark theme for application
- **Multi-level Organization**: Areas → Projects → Tasks hierarchy
- **Authentication**: Supabase Auth with protected routes
- **Sharing**: Projects can be shared via permanent links

## Target Users

Teams and organizations needing project management with document storage and collaborative task tracking.
